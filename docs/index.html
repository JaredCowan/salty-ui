Designed for everyone, everywhere.

Salty-UI makes front-end web development faster and easier. It's made for folks of all skill levels, devices of all shapes, and projects of all sizes.

Get Started

One framework, every device.

Salty-UI easily and efficiently scales your websites and applications with a single code base, from phones to tablets to desktops with CSS media queries.

Full of features

With Salty-UI, you get extensive and beautiful documentation for common HTML elements, dozens of custom HTML and CSS components, and awesome jQuery plugins.

Salty-UI is open source.

It's hosted, developed, and maintained on GitHub.
